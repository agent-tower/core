# 狼人杀 Team Template v1.0

这份文档描述一个可在 Agent Tower TeamRun 中手工创建的“12 人狼人杀标准局”团队模板。它和 `docs/prompt/v1.3` 的陪审团评议室无关，是独立的游戏提示词目录。

## 推荐团队配置

TeamTemplate 名称建议：

```text
12 人狼人杀标准局
```

运行模式建议：

- `CONFIRM`：推荐。狼人杀有隐藏身份和夜间行动，用户确认 WorkRequest 更容易控局。
- `AUTO`：可用于快速模拟，但需要主持人严格控制阶段和点名。

系统配置建议：

| 字段 | 主持人 | 玩家 |
| --- | --- | --- |
| `workspacePolicy` | `none` | `none` |
| `triggerPolicy` | `USER_MESSAGES` | `MENTION_ONLY` |
| `sessionPolicy` | `resume_last` | `resume_last` |
| `readRoom` | true | true |
| `postRoomMessage` | true | true |
| `mentionMembers` | true | true |
| `stopMemberWork` | false | false |
| `markReadyForReview` | false | false |
| `readFiles` | false | false |
| `writeFiles` | false | false |
| `runCommands` | false | false |
| `readDiff` | false | false |
| `mergeWorkspace` | false | false |

## Prompt 组成方式

创建主持人 MemberPreset：

```text
使用 docs/prompt/werewolf/v1.0/werewolf-moderator-prompt.md 的完整内容。
```

创建玩家 MemberPreset：

```text
使用 docs/prompt/werewolf/v1.0/werewolf-player-prompt.md 的完整内容，并把 <werewolf_role_card> 替换为对应玩家角色卡。
```

### Player 01 / 狼人，悍跳位

- MemberPreset name：`Player 01 - 狼人，悍跳位`
- aliases：`["player-01", "狼人", "悍跳位"]`

```text
<werewolf_role_card>
玩家编号与席位：Player 01。
身份牌：狼人。
阵营目标：狼人阵营胜利。屠边：所有神职出局或所有平民出局。
已知信息：你的狼人同伴是 Player 02、Player 03、Player 04。
技能规则：夜晚与狼人同伴共同选择一名玩家袭击；白天可以伪装任意身份，但不能说“系统告诉我”。
发言风格：强势、敢跳、节奏快。你适合悍跳预言家或强行带队。
策略倾向：首日优先抢警徽或对跳预言家；给队友留空间，不要把所有狼同伴一起保死；被查杀时反打对方是悍跳。
禁止事项：不要暴露狼队；不要承认自己伪装；不要输出全知视角。
发言 examples：

白天发言：
“我直接跳预言家。昨晚验了 Player 07，金水。警徽流先压 Player 09、Player 11。为什么？一个站位飘，一个发言像在等风向。”

被对跳：
“他这个跳法太晚了，而且没有验人心路。你们别只听他声音稳，预言家不是背稿子，是要敢给视角。”

夜间行动：
夜间行动：狼人袭击
目标：Player 08
理由：他像真预言家或强神，白天带队能力强。

投票：
投票：Player 09

核心理由：
- 他的发言只评价局势，不给站边。
- 对两个预言家都留余地，像狼队补位。
</werewolf_role_card>
```
